# AI-Driven Drone Delivery: Transforming the Future of Logistics

An interactive educational webapp exploring the use of AI in drone-based delivery systems and their impact on the logistics industry in the United States (2025-2035).

## Project Overview

This webapp provides an engaging, interactive experience to learn about:
- The evolution and timeline of drone delivery technology
- Key benefits including speed, cost savings, and environmental impact
- Technical and regulatory challenges facing the industry
- Interactive cost calculator comparing traditional vs. drone delivery
- Knowledge quiz to test understanding

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Scrolling timeline of drone delivery evolution
  - Cost calculator with real-time calculations
  - 5-question interactive quiz with explanations
  - Smooth scrolling navigation
  - Progress bar showing scroll position
- **Educational Content**:
  - Comprehensive overview of AI-driven drone delivery
  - Comparison table: Traditional vs. Drone delivery
  - Benefits and challenges analysis
  - Future outlook and projections

## Technologies Used

- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript (no external dependencies)
- Modern web standards and best practices

## Deployment Instructions

### Option 1: GitHub Pages (Recommended)

1. **Create a new GitHub repository**
   ```bash
   git init
   git add index.html README.md
   git commit -m "Initial commit: AI-Driven Drone Delivery webapp"
   ```

2. **Push to GitHub**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click Settings → Pages
   - Under "Source", select "main" branch
   - Click Save
   - Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

### Option 2: Quick Deploy Services

**Netlify**:
1. Drag and drop the `index.html` file to [Netlify Drop](https://app.netlify.com/drop)
2. Your site will be instantly deployed with a random URL

**Vercel**:
1. Install Vercel CLI: `npm install -g vercel`
2. Run `vercel` in your project directory
3. Follow the prompts

### Option 3: Local Testing

Simply open `index.html` in any modern web browser to view the webapp locally.
